# 🎨 Anmol Studio (Full Stack E-Commerce)

🌐 **Live Demo:** https://anmol-studio.vercel.app/  

---

## 📖 About the Project  
💡 **Anmol Studio** is a full-stack e-commerce web application with a modern UI and complete backend functionality.  
It allows users to browse products, add to cart, place orders, and manage accounts, while also providing an admin panel for product and order management.

---

## ✨ Features  

### 🛍️ User Features  
- 🔐 User authentication (Login/Signup)  
- 🛒 Add to cart & manage cart  
- 📦 Place orders  
- 📜 View order history  
- 🔍 Search & browse products  

### 🛠️ Admin Features  
- ➕ Add / update / delete products  
- 📦 Manage orders  
- 🖼️ Image upload using Cloudinary  

### 🎨 UI Features  
- 📱 Fully responsive design  
- ⚡ Fast and smooth performance  
- 🎯 Clean and modern UI

## ⚙️ Installation & Setup  

### 1️⃣ Clone Repository  
git clone https://github.com/gitwithanmoldubey/Anmol-Studio.git  
cd Anmol-Studio  

---

### 2️⃣ Setup Backend  
cd backend  
npm install  
npm run server  

---

### 3️⃣ Setup Frontend  
cd frontend  
npm install  
npm run dev  

---

### 4️⃣ Setup Admin Panel  
cd admin  
npm install  
npm run dev
